# Eggverse: Underwater Merge

This repository contains the source code for my Unity 2D mobile game, **Eggverse: Underwater Merge**.

In this relaxing underwater merge game, players combine matching eggs to create larger, more valuable eggs while earning points and chasing a new high score. The goal is simple: keep merging for as long as possible without running out of space.

The game is available as a free download on **Google Play** and the **Apple App Store**.

## Features

* Simple and engaging merge gameplay
* Offline play—no internet connection required
* Local high score saving
* Designed for Android and iOS

## Privacy Policy

This repository also includes the game's Privacy Policy, which complies with the requirements for distribution on the Google Play Store and the Apple App Store.

## Built With

* Unity (2D)
* C#
